# To Do List Web App

Este projeto é uma aplicação web simples de gerenciamento de tarefas (To Do List), inspirada em ferramentas como o Microsoft To Do, com foco em funcionalidade e usabilidade.

## Funcionalidades

- Adicionar novas tarefas
- Listar tarefas dinamicamente
- Definir prioridade:
  - Baixa (verde)
  - Média (amarelo)
  - Alta (vermelho)
- Marcar tarefas como concluídas
- Remover tarefas
- Interface limpa e intuitiva

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Layout

A interface é simples e funcional, com:
- Campo de entrada para tarefas
- Seletor de prioridade
- Lista dinâmica de tarefas
- Destaque visual por prioridade
